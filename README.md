TwitterGlobe
============

Globe visualization of tweets

## Configuration
Create a *configuration.json* file with the following information:

    {
      "consumer_key":         "...",
      "consumer_secret":      "...",
      "access_token":         "...",
      "access_token_secret":  "..."
    }

**Note**: The server runs on port 80, change it in the *app.js* file.
